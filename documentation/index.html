HTML Structure General layout and content structure

Streamline is a Responsive one page portfolio which is mobile and retina display ready. There are 6 main sections I want to summarize here. There is the navbar, the team portraits, the services, the gallery items, the pricing and the testimonials.

Navigational Bar

The navigation is a fixed header that appears after scrolling past the welcome section. It will shrink to a dropdown menu on top on smaller devices.

To change links in the menu, change the li tags in ul tag with the nav class.

<ul class="nav navbar-nav">
  <li><a href="#about"><span class="glyphicon glyphicon-user"></span>Team</a></li>
  <li><a href="#services"><i class="icon-gear"></i>Services</a></li>
  <li><a href="#work"><span class="glyphicon glyphicon-briefcase"></span>Work</a></li>
  <li><a href="#pricing"><i class="icon-dollar"></i></span>Pricing</a></li>
  <li><a href="#clients"><i class="icon-comments"></i></span>Clients</a></li>
</ul>

Services

The services section consists of 4 circles that represent different services one could offer. They also have a tooltip that appears on hover.

You can change the title of the circle in thespan tag, and also change the tooltip text in the title field of the icon.

<div class="col-md-3 col-sm-3 col-xs-6">
  <div class="service-icon">
    <i class="icon-sitemap" data-toggle="tooltip" title="Professional application planning, including mockups, wireframes, and designs."></i>
    <span>PLANNING</span>                
  </div>
</div>

Portfolio Items

The portfolio uses MixItUp to sort and filter categories of items. To learn how to use that check the javascript section.

To change the portfolio image change the img tag. You can also change the link in the a tag 

<li class="col-md-4 col-sm-4 col-xs-12 mix category_1" data-cat="1">
  <a href="#" class="mix-cover">
    <img src="img/sintel-bedroom.png">
    <span class="cover"><h2>Sintel Bedroom</h2></span>
  </a>
</li>

To change the price change the h2 tag. You can edit the features of each option in the tbody.

JavaScript JavaScript customization and highlights

× Bootstrap Javascript: Bootstrap comes with its own javascript file that you should never touch or change. It contains all the scripts for the bootstrap framework. I have also included that minimized version.

JavaScript Files

Only edit init.js, the rest of the files are plugins or libraries that shouldn't be edited

Tag Description
init.js This is the custom javascript that initializes everything like the carousels, the horizontal scrolling and the cover animations.
jquery-1.9.1.min.js jQuery is a Javascript library that greatly reduces the amount of code that you must write.
bootstrap.js This is the Bootstrap framework javascript that enables all of its functionalities. Do not edit this.
jquery.mixitup.js This is jQuery plugin that enables the portfolio sorting and categorization.

MixItUp Portfolio / Gallery

The gallery uses a jQuery plugin called mixitup where you can tag items with certain categories which you can then hide and show depending on specific categories.

You can assign categories to your gallery items by adding a category name as a class to the li tags with class mix

For the button filters, you can add data-filters that correspond to classes you assign gallery items.

      <div class="controls">	
  <a class="btn filter" data-filter="all">SHOW ALL</a>
  <a class="btn filter" data-filter="category_1">CATEGORY 1</a>
  <a class="btn filter" data-filter="category_2">CATEGORY 2</a>
  <a class="btn filter" data-filter="category_3">CATEGORY 3</a>
  <a class="btn filter" data-filter="category_1 category_3">CATEGORY 1 &amp; 3</a>
</div>

Parallax Backgrounds

Besides the first background image, you can also set content sections to have custom background images with parallax scrolling.

To do this, add the class image to the desired section. Then you can edit the css file and add a background image to that sections background class.

<div id="services" class="section image">
  <div class="background parallax"></div>
</div>

#services .background {
  background: url("../img/office.jpg") fixed no-repeat; 
}
